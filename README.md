# ftp-security-testing-script
This project is a beginner level cybersecurity script that illustrate a brute-force (dictionary-based) password attack method against an FTP server. It uses Python’s ftplib module to attempt logging into an FTP server by trying multiple passwords from a user-provided wordlist file. Moreover, The program continues testing each password until a successful login is found or the list is exhausted.

IMPORTANT NOTE-
This project is created strictly for educational and ethical cybersecurity learning purposes. Any misuse of this tool for unauthorized access is illegal and against ethical guidelines. Therefor, The script must only be used on systems for which explicit authorization has been granted. Unauthorized use is strictly prohibited and may be illegal.
